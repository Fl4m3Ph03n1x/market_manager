    View Source Manager.Impl.Strategy behaviour (Market Manager v2.1.5)

    Behaviour that specifies the functions any strategy must implement. It is mostly used to provide an implementation of the Strategy pattern from OO languages, but adapted to Elixir.

    The pricing strategy will be selected and executed dynamically at runtime, as long as it implements this behaviour.
