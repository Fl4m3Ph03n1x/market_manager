Market Manager
v2.1.8

      View Source AuctionHouse (Market Manager v2.1.8)

      Library representing the interface for the auction house. Responsible for making calls and decoding the answers from the auction house into a format the manager understands.

      Summary

      Functions

      delete_order(order_id)

      Deletes an order from warframe market.

      get_all_orders(item_name)

      Gets all warframe market orders for the item with the given name. The item's name is in human readable format. This function also converts the name into a format that the external party can understand.

      login(credentials)

      Stores the user's credentials and authenticates with the auction house to make requests. Must be invoked every time the application is launched. It also performs the necessary steps for authorization. Returns user information.

      logout()

      Deletes the current session and user data from the this application. Does not interact with the External AuctionHouse, this is a local operation only.

      place_order(order)

      Places an order in warframe market.

      recover_login(auth, user)

      Feeds the authorization information directly to the AuctionHouse. Used when the login data is being recovered from a past login. Will only fail if this service is down.
