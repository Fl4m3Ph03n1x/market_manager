    Data about the Request or Response being made. Includes information important for 3rd party modules to use but that is not related to the arguments of the flow itself.

    Used mostly to notify interested parties of the results of the flow. When the flow is done, the "send?" field should be set to true so the handler know it has to send the results back to the interested parties.
